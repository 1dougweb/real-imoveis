Visão Geral do Sistema

O sistema Laranja Real Imóveis é uma aplicação web completa para gestão imobiliária, desenvolvida com Laravel (backend) e React (frontend). A aplicação permite o gerenciamento de imóveis, clientes, visitas, contratos e transações financeiras.

Arquitetura

O sistema segue uma arquitetura de cliente-servidor, com separação clara entre frontend e backend:

  • Backend: API RESTful desenvolvida com Laravel 10, seguindo os princípios de APIs modernas e seguras.
  • Frontend: Aplicação SPA (Single Page Application) construída com React 18 e TypeScript, utilizando hooks e componentes funcionais.
  • Autenticação: Sistema baseado em JWT (JSON Web Tokens) para autenticação e autorização de usuários.
  • Banco de Dados: MySQL 8 com migrações e seeders para fácil instalação e atualização.

Fluxos de Usuário

O sistema suporta diferentes tipos de usuários e fluxos de navegação:

  • Clientes: Podem procurar imóveis, agendar visitas e acompanhar propostas/contratos.
  • Corretores: Gerenciam visitas, clientes e imóveis, além de criar propostas e contratos.
  • Gerentes: Supervisionam todas as operações, incluindo financeiras e relatórios.
  • Administradores: Gerenciam todos os aspectos do sistema, incluindo usuários e configurações.
